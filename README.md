# Color Flipper 🎨

A simple web app that changes the background color of the page using buttons.  
Built with **HTML**, **CSS**, and **JavaScript**.

## Features

- Change background color to **Red**, **Green**, or **Blue**
- Generate a **Random color** using RGB values
- Starts with a random color on page load

---

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (ES6)
